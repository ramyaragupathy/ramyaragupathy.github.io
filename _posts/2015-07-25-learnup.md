**OpenStreetmap** 
  - Login to www.osm.org
  - Tracing with iD editor
    - Open http://tasks.openstreetmap.in/project/25
    - Select a grey block to map and click `Start Mapping`
    - Select `Edit with iD editor`
    - A brief on how to edit with iD editor
      - Zoom to a missing road.Trace it using `line` tool and tag it.
      - Trace buildings using `area` tool.  
      - Using `ctrl+z` to undo any changes
      - Finally Save changes with an appropriate comment.
    - If there is no more missing data in the area, mark the task as `done` in tasking manager.

  - Neighbourhood mapping using JOSM
    - Download building tool plugin
    - Download the neighbourhood area in JOSM
    - A brief on basic tools
      - Using the satellite imagery options
      - Tracing roads and tagging it appropriately
      - Tracing buildings using the plugin
    - Adding nodes, street names, house numbers and building layers
    - Uploading the changes with an appropriate comment

**Mapbox Studio** 
- Open Mapbox Studio
- Sign into Mapbox and authorize
- Create a new `basic` style project
- `Save` your project
- Explore the menu options and the documentation
- Change the values of `@land` and `@water` to a different color and save
- To export your map as an image, click the export button from the top right
- Upload the map online using `Settings>Upload to Mapbox` option. Give an appropriate name and description.
- Now click on your mapid. This will take to your account page. Click the view map link.

**LOTR Theme**

- Download custom images for land, water, hills from [dropbox](https://www.dropbox.com/s/2jk7jss48uxg2nb/Learnup.zip?dl=0)

- Extracting separate dataset for important peaks of the world using [Overpass turbo](http://overpass-turbo.eu/).
  - Overpass turbo
    - Go to http://overpass-turbo.eu/
    - Open the `query wizard` from the menu.
    - Build and run the query
    - Export the data as a .geojson file

- Creating a new blank project with peaks dataset as the source.

- Inserting custom images for land, water, hills and green areas.
